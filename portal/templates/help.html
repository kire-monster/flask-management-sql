Comandos de ayuda

SQL Server

Mostrar la informacion de las tablas: SELECT * FROM [INFORMATION_SCHEMA].[TABLES]
o las vistas: SELECT * FROM [INFORMATION_SCHEMA].[VIEWS]

Mostrar los procedimientos: SELECT Nombre=([SPECIFIC_SCHEMA] + '.' + [ROUTINE_NAME]) FROM INFORMATION_SCHEMA.ROUTINES WHERE [ROUTINE_TYPE] = 'PROCEDURE'

o Funciones SELECT Nombre=([SPECIFIC_SCHEMA] + '.' + [ROUTINE_NAME]) FROM INFORMATION_SCHEMA.ROUTINES WHERE [ROUTINE_TYPE] = 'FUNCTION'

Ver la informacion de la tabla select * from INFORMATION_SCHEMA.COLUMNS WHERE TABLE_NAME = 'tutabla'
